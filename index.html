Riddle Game

About The Project

This application uses the Flask micro web framework to allow users to play a riddle game. The aim of the game is for the user to answer riddles displayed to them on screen. The user must first enter their name to play. Subsequently, riddles are displayed one at a time on the screen with an answer box appearing below. The user can only progress to the next riddle upon submission of a correct answer. There user must answer 11 riddles in total to complete the game. Each correct answer increments their score by 1 point. If the user decides they cannot finish the game, they can quit early to view the high scores leaderboard.

Project Link
http://python-riddle-game-ci.herokuapp.com/
